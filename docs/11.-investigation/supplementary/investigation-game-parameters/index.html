8. Parameter Relationships

Critical Dependencies

Tuning Priorities

If metrics are off, adjust in this order:

  1. Parameters (hours): Word counts, timing, choice counts
  2. Presentation (days): Narrator voice, framing, pacing
  3. Content (days): Case details, explanation clarity, reflection prompts
  4. Structure (weeks): Act balance, pivot placement, revelation timing
  5. Mechanics (weeks): Choice types, adaptation logic, branching structure
  6. Dynamics (major): Fundamental approach to what players do

10. Implementation Notes

State Persistence Requirements

Must track across sessions:

  • Current node position
  • All choices made (for callbacks)
  • Player state variables (skepticism, engagement, etc.)
  • Time spent per section
  • Depth content viewed

A/B Testing Candidates

If metrics underperform, test:

  • Hook variants (different opening lines)
  • Case 3 detail level (more/less background)
  • Revelation pacing (faster/slower)
  • Reflection prompt wording (more/less direct)
  • Fourth-wall break frequency (more/fewer)
  • Narrator directness (more/less challenging)
