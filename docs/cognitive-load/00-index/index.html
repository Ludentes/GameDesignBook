Cognitive Load and Game System Design

Chapter Overview

This chapter explores cognitive load as a fundamental constraint in game design and a powerful tool for creating specific experiential scaffolds. We examine how different cognitive load profiles create different play experiences, and provide a systematic framework for designing game systems that match their intended experiential goals.
