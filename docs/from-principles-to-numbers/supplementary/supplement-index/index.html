Balance Chapter - Supplementary Materials Index

Overview

The Balance chapter (Chapter 5) focuses on systematic approaches and frameworks. These supplements provide the detailed reference material, formulas, and protocols that would bloat the main chapter.

Main Chapter: Conceptual frameworks, decision processes, worked examples Supplements: Lookup tables, formulas, detailed protocols

How to Use These Supplements

Stage 1: Initial Design (Before Implementation)

Use: Supplement 1 (Measurables Catalog)

  • Look up your aesthetic + paradigm combination
  • Identify relevant measurables for your game type
  • Set initial target values
  • Use as starting points for parameters

Example: "I'm making a Challenge aesthetic boss fight game with Consistency Balance" → Look up Challenge + Consistency section → Find boss fight measurables (TTK: 2-5 minutes, attempts: 8-15, etc.) → Set these as initial targets

Stage 2: Parameter Setting (Implementation)

If <20 entities:

  • Hand-tune each using Chapter 5 main approach
  • Reference Supplement 1 for target ranges

If 20-100 entities:

  • Use Supplement 2 (Formula Templates)
  • Apply scaling formulas
  • Validate samples with playtesting

Example: "I'm balancing 30 enemy types across 20 levels" → Use HP scaling formula from Supplement 2 → Test at levels 1, 5, 10, 15, 20 → Adjust formula if needed

Stage 3: Validation (Playtesting)

Use: Supplement 3 (Playtest Guide)

  • Follow paradigm-specific testing protocol
  • Track measurables using provided templates
  • Identify problems using diagnostic patterns
  • Iterate using decision trees

Example: "Testing Consistency Balance boss fight" → Use Consistency testing protocol from Supplement 3 → Track learning curve, TTK, pattern recognition → If TTK too long → Use decision tree to adjust

Stage 4: Iteration (Refinement)

Use all three together:

  • Supplement 1: Check if targets are still appropriate
  • Supplement 2: Recalculate formulas with new parameters
  • Supplement 3: Test again, track improvements

Continue until:

  • Measurables within ±15% of targets (Supplement 1)
  • Formulas produce consistent values (Supplement 2)
  • Playtests show positive experience (Supplement 3)

Integration with Main Chapter

Chapter 5 teaches you:

  • What balance is and why it matters
  • How aesthetics require specific dynamics
  • The three balance paradigms
  • How to choose the right paradigm
  • How to derive parameters from measurables
  • When to scale and when you need help
