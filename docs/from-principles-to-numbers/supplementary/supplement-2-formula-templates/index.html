Balance Supplement 2: Formula Templates and Scaling Systems

Document Purpose

This supplement provides mathematical formulas and systematic approaches for parameter scaling when you have 20+ entities to balance (enemies, items, abilities, levels).

When to use formulas:

  • 5-20 entities: Hand-tune each (Chapter 5 main approach)
  • 20-100 entities: Use formulas with sample validation
  • 100+ entities: Need procedural tools (beyond this book's scope)

What this supplement provides:

  • HP/Damage scaling formulas
  • Resource economy formulas
  • Progression curve calculators
  • Cost-benefit ratio tools

Part 1: HP and Damage Scaling Formulas

Basic Linear Scaling

Use when: Simple progression, consistent difficulty curve

Enemy HP Formula:

Example (GrowthRate = 0.15):

  • Level 1: 100 HP
  • Level 5: 160 HP (100 × 1.6)
  • Level 10: 250 HP (100 × 2.5)
  • Level 20: 400 HP (100 × 4.0)

Pros: Simple, predictable, easy to tune Cons: Can feel too gradual or too steep

Tuning the formula:

  • GrowthRate too high → Late game feels impossible
  • GrowthRate too low → Late game feels trivial
  • Test at levels 1, 5, 10, 15, 20 (key breakpoints)

Exponential Scaling

Use when: Accelerating difficulty curve, power fantasy emphasis

Enemy HP Formula:

Example (GrowthMultiplier = 1.10):

  • Level 1: 100 HP
  • Level 5: 161 HP (100 × 1.10^5)
  • Level 10: 259 HP (100 × 1.10^10)
  • Level 20: 673 HP (100 × 1.10^20)

Pros: Accelerating growth feels like real progression Cons: Can explode out of control if not careful

Warning: Exponential growth requires careful testing. If GrowthMultiplier > 1.15, late game becomes unplayable quickly.

Player Damage Scaling

Key principle: Player damage should scale slightly faster than enemy HP to create power fantasy

Formula:

Example:

  • Enemy HP GrowthRate: 0.15
  • Player Damage Rate: 0.20
  • Result: Player kills enemies progressively faster (power fantasy)

Time-to-Kill (TTK) Formula:

Target TTK by game type:

  • Action game trash mob: 2-5 seconds
  • Action game elite: 8-15 seconds
  • Action game boss: 120-300 seconds (2-5 minutes)
  • RPG trash mob: 5-10 seconds
  • RPG elite: 15-30 seconds
  • RPG boss: 180-600 seconds (3-10 minutes)

Validating Scaling Formulas

Test at key breakpoints:

  1. Level 1 - Tutorial experience, should be easy
  2. Level 5 - Early game, players learning mechanics
  3. Level 10 - Mid game, players comfortable
  4. Level 15 - Late mid game, mastery expected
  5. Level 20 - Endgame, full challenge

For each level, measure:

  • Actual TTK (does it match targets?)
  • Player success rate (too easy/hard?)
  • Player feedback (feels appropriate?)

Adjustment process:

  1. If Level 10 too hard: Reduce GrowthRate by 0.05
  2. If Level 20 too easy: Increase GrowthRate by 0.05
  3. If entire curve wrong: Change BaseHP or BaseDamage
  4. Re-test and iterate

Part 7: Common Formula Problems

Problem: Exponential Explosion

Symptom: Late game becomes impossibly hard

Cause: GrowthMultiplier too high or too many multipliers stacking

Fix:

Rule: If using exponential, keep multiplier < 1.15

Problem: Linear Boring

Symptom: Late game feels same as early game

Cause: GrowthRate too low

Fix:

Rule: Aim for 3-5× growth from start to end

Problem: Breakpoint Discontinuity

Symptom: Difficulty spike at breakpoint

Cause: Piecewise formula with bad transition

Fix:

Rule: Breakpoint rates should differ by <0.10
