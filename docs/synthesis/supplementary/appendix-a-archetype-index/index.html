Appendix A: Archetype Library - Index

Complete Archetype Collection

This appendix contains 10 fully detailed game archetypes. Each archetype is provided as a separate file for easy reference.

The 10 Archetypes

  1. Daily Puzzle Game (16KB)

    • Wordle-style daily challenge games
    • Compressed loops, casual players, 5-30 min sessions
    • Primary aesthetics: Challenge + Completion + Discovery

  2. Roguelike (3KB - summary)

    • Permadeath with meta-progression
    • Full stack loops, hardcore players, 20-60 min runs
    • Primary aesthetics: Challenge + Mastery + Discovery

  3. 4X Strategy (5KB)

    • Civilization-style grand strategy
    • Macro-heavy, hardcore players, 2-8 hour sessions
    • Primary aesthetics: Mastery + Power + Discovery

  4. Cozy Life Sim (6KB)

    • Animal Crossing-style relaxation games
    • Flexible loops, all skill levels, 20min-4hr sessions
    • Primary aesthetics: Expression + Discovery + Fellowship + Collection

  5. One-Shot TTRPG (6KB)

    • Single-session tabletop games
    • Three-act narrative, adults, 3-5 hour sessions
    • Primary aesthetics: Narrative + Fellowship + Discovery

  6. Metroidvania (5KB)

    • Exploration with ability-gating
    • Full stack loops, skilled players, 1-3 hour sessions
    • Primary aesthetics: Discovery + Mastery + Power + Challenge

  7. Idle/Incremental (5KB)

    • Cookie Clicker-style passive games
    • Inverted macro-heavy, all players, 30sec-2hr sessions
    • Primary aesthetics: Power + Mastery + Completion

  8. Deck-Building (5KB)

    • Slay the Spire-style card games
    • Full stack loops, strategy fans, 20-60 min sessions
    • Primary aesthetics: Mastery + Challenge + Discovery + Expression

  9. Walking Simulator (5KB)

    • Environmental storytelling games
    • Linear narrative, story-focused, 1-5 hour complete experience
    • Primary aesthetics: Narrative + Discovery + Wonder + Reflection

  10. Auto-Battler (5KB)

    • Teamfight Tactics-style strategy
    • Full stack with elimination, strategy fans, 30-60 min games
    • Primary aesthetics: Mastery + Challenge + Discovery + Competition

How to Use These Archetypes

Each archetype file contains:

  • Overview and core appeal
  • Primary aesthetics
  • Player profile match
  • Complete temporal structure
  • Common mechanics
  • Detailed resource patterns
  • Content needs and formulas
  • Platform considerations
  • Monetization patterns
  • Social features
  • Success metrics
  • Common variations
  • Design pitfalls to avoid
  • Example games
  • When the archetype fails

Using Archetypes in the Framework:

  • Phase 0: Browse archetypes to find closest match to your concept
  • Phase 1: Use archetype's aesthetic profile as starting point
  • Phase 2: Adopt or adapt archetype's temporal structure
  • Phase 3: Reference archetype's common mechanics and resources
  • Phase 4: Use archetype's content formulas and platform guidance
  • Phase 5: Check against archetype's common pitfalls

Modification Guidelines:

  • Keep what serves your specific aesthetic goals
  • Change what doesn't fit your player profile or constraints
  • Document why you're diverging from the archetype
  • Test modified patterns more thoroughly than pure archetypes

Quick Reference: Find Your Archetype

By Session Length

  • Very Short (5-30 min): Daily Puzzle, Idle/Incremental (check-ins)
  • Short (20-60 min): Roguelike, Deck-Building, Auto-Battler
  • Medium (1-3 hours): Metroidvania, Walking Simulator, Cozy Life Sim
  • Long (2-8 hours): 4X Strategy
  • Complete (3-5 hours one-shot): One-Shot TTRPG, Walking Simulator

By Player Type

  • Casual: Daily Puzzle, Cozy Life Sim, Idle/Incremental
  • Midcore: Deck-Building, Auto-Battler, Metroidvania
  • Hardcore: Roguelike, 4X Strategy
  • Story-Focused: Walking Simulator, One-Shot TTRPG

By Primary Aesthetic

  • Challenge: Daily Puzzle, Roguelike, 4X Strategy, Metroidvania, Deck-Building, Auto-Battler
  • Mastery: Roguelike, 4X Strategy, Metroidvania, Idle/Incremental, Deck-Building, Auto-Battler
  • Discovery: Daily Puzzle, Roguelike, 4X Strategy, Cozy Life Sim, One-Shot TTRPG, Metroidvania, Deck-Building, Walking Simulator, Auto-Battler
  • Narrative: One-Shot TTRPG, Walking Simulator
  • Expression: Cozy Life Sim, Deck-Building
  • Power: 4X Strategy, Metroidvania, Idle/Incremental

By Platform

  • Mobile-First: Daily Puzzle, Idle/Incremental, Auto-Battler
  • PC-Primary: 4X Strategy, Roguelike, Deck-Building
  • Console-Friendly: Metroidvania, Cozy Life Sim, Walking Simulator
  • Tabletop: One-Shot TTRPG

By Replayability

  • High Replay: Roguelike, 4X Strategy, Idle/Incremental, Deck-Building, Auto-Battler
  • Moderate Replay: Daily Puzzle, Metroidvania, Cozy Life Sim
  • Low Replay (Single Experience): Walking Simulator, One-Shot TTRPG
